### Basic Data Types
```elixir
255       # integer
0b0110    # binary
0o644     # octal
0x1F      # hexadecimal
3.14      # float
1.0e-10   # float (support e for exponencial values)
true      # booleans
:foo      # atom (constant whose name is its value)
"Hello"   # strings
```
